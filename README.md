# 📱 Interactive Story Viewer

An interactive Instagram-style story viewer built using **HTML**, **CSS**, and **JavaScript**.  
This project dynamically displays user stories and allows users to view them in fullscreen with smooth UI interaction.

---

## ✨ Features

- Instagram-style stories UI
- Dynamic story rendering using JavaScript
- Clickable story thumbnails
- Fullscreen story preview
- Automatic story close after a few seconds
- Smooth and clean user interface
- Horizontal scrollable stories section

---

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript (Vanilla JS)

---

## 📂 Project Structure

```bash
Interactive-Story-Viewer/
│
├── index.html
├── style.css
└── script.js
```

---

## 🚀 How It Works

- Story data is stored inside a JavaScript array.
- JavaScript dynamically creates story elements.
- Clicking a story opens it in fullscreen mode.
- The story automatically disappears after 2 seconds.

---

## 🧠 Core Concepts Used

- DOM Manipulation
- Event Listeners
- Array Iteration (`forEach`)
- Dynamic HTML Rendering
- CSS Positioning
- Overflow Scrolling
- Fullscreen UI Effects
